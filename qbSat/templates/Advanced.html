The default username and password for both the qbSat Wi-Fi and qbSat user on the Pi is "qbSat" and "raspberry". If you want to modify the code on the Pi, you'll want to use SSH and sign in as the user qbSat. If you want the Raspberry Pi to also have access to the internet for development, you'll need to plug in an additional Wi-Fi adapter to a USB port and edit the wpa_supplicant.conf file to match your Wi-Fi login.
